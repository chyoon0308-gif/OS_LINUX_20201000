- 실습 일시: 2026-04-17

## 1. 실습 목표
- 파일시스템 기본 정보 확인 (df, diskutil)
- inode, 하드링크, 심볼릭링크 이해
- 블록 할당 및 낭비 크기 분석
- Docker 파일시스템 구조 확인

## 2. 주요 실습 내용

### 파일시스템 확인
- `df -h` : 디스크 사용량 확인
- `diskutil list` : 블록 디바이스 구조 확인

### 링크 실습
- 하드링크: 원본 삭제 후에도 데이터 유지
- 심볼릭링크: 원본 삭제 시 dangling link 발생

### 블록 할당 분석
- 1KB 파일도 4KB(1블록) 점유 → 3KB 낭비
- 40KB 이상은 낭비 없이 할당

### Docker 파일시스템
- Storage Driver: overlayfs
- 장기 운영 시 로그 무제한 증가 주의
